OptimusKG integrates data from 12 external biomedical data sources. Each source is processed in the [bronze layer](/docs/architecture/medallion-layers) by dedicated node functions.

## Source Overview

| Source | Type | Description |
| --- | --- | --- |
| [OpenTargets](/docs/data-sources/opentargets) | Targets, Diseases, Drugs | Platform for target identification and validation |
| [DrugBank](/docs/data-sources/drugbank) | Drugs, Proteins | Comprehensive drug and drug-target database |
| [DrugCentral](/docs/data-sources/drugcentral) | Drugs, Diseases | Drug information resource |
| [CTD](/docs/data-sources/ctd) | Exposures | Comparative Toxicogenomics Database |
| [DisGeNET](/docs/data-sources/disgenet) | Diseases, Genes | Disease-gene association database |
| [Bgee](/docs/data-sources/bgee) | Gene Expression | Gene expression in anatomy |
| [OnSIDES](/docs/data-sources/onsides) | Side Effects | Drug side effects from FDA labels |
| [Reactome](/docs/data-sources/reactome) | Pathways | Biological pathway database |
| [Ontologies](/docs/data-sources/ontology) | Ontologies | GO, HP, MONDO, UBERON |
| [PPI](/docs/data-sources/ppi) | Interactions | Protein-protein interactions |
| [Gene Names](/docs/data-sources/gene-names) | Nomenclature | HGNC gene nomenclature |

## Data Flow

Each data source follows the same processing pattern:

1. **Landing**: Raw data downloaded via [Providers](/docs/architecture/providers)
2. **Bronze**: Parsed and standardized into typed Polars DataFrames
3. **Silver**: Merged with other sources into unified entity and relationship tables

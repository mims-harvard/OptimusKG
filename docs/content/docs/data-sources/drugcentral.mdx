[DrugCentral](https://drugcentral.org/) provides drug information including indications, pharmacology, and regulatory data.

## Bronze Nodes

Processes drug and disease information from DrugCentral's SQL dump distribution.

## Dataset Type

Uses the [`SQLDumpQueryDataset`](/docs/architecture/datasets) to load the SQL dump into a temporary database and execute queries.
